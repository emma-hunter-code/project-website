About Me

Code, data, design, cats.

Code

I learnt to code with SheCodes from Jan-Nov 2025, using HTML, CSS, Bootstrap/Responsive Design in VS Code. I have built applications using JavaScript including React.js, Python, APIs, and AI tools. I manage code via my GitHub account and host SEO-friendly sites on Netlify.


Before I learnt to code, I assumed I did not have the right skills to be a Developer - but I got curious when a colleague suggested it might suit me, and it's since become something I love! I get a lot of satisfaction from building practical, useful websites and applications.

Work

I've worked in Research Ops since 2009, developing processes from the ground up through to implementation. I do a lot of data handling and process/sytem development and implementation (See 'Data'). A favourite work project was designing and implementing a management system for multiple teams using REDCap. That is where my interest in coding began. Take a closer look at my work experience and skills on my LinkedIn profile.

Data

Design

Personal
